# Dashboard Senac

## Sobre
> Projeto desenvolvido para o curso de informática em internet do Senac ES 2023.

## 🦾 Funcionalidades
---
- Login via QRcode
- Calendário
- Agenda editável
- Cronômetro
- Notas autoadesivas
- Tarefas em kanban board

## 🪄 Features
---
- Responsividade
- Modo Noturno

## 🛠 Tecnologias utilizadas
---
- Figma
- HTML5
- CSS3
- JavaScript
